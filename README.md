# File Collector

指定した期間内に更新されたファイルを収集するElectronアプリケーションです。

## 機能

- **フォルダ選択**: 監視対象フォルダと収集先フォルダを選択
- **期間指定**: 過去N時間以内またはカスタム日時以降に更新されたファイルを検索
- **除外設定**: 特定のフォルダを除外（ワイルドカード対応）
- **ファイル選択**: 見つかったファイルの中から収集対象を個別に選択
- **階層保持**: フォルダ構造を保ったままファイルをコピー（オプション）
- **プログレス表示**: ファイル収集の進行状況をリアルタイム表示

## スクリーンショット

アプリケーションの主な画面：
- フォルダ選択画面
- ファイルスキャン結果表示
- ファイル収集進行状況

## インストールと実行

### 必要な環境
- Node.js (v14以上推奨)
- npm

### セットアップ

1. リポジトリをクローン
```bash
git clone <your-repository-url>
cd file_collecter
```

2. 依存関係をインストール
```bash
npm install
```

3. アプリケーションを起動
```bash
npm start
```

開発モードで起動する場合:
```bash
npm run dev
```

## 使用方法

1. **監視フォルダを選択**: 更新されたファイルを検索するフォルダを選択
2. **収集先フォルダを選択**: ファイルをコピーする先のフォルダを選択
3. **検索条件を設定**:
   - 期間指定（過去N時間）またはカスタム日時を設定
   - 必要に応じて除外フォルダを指定
4. **ファイルをスキャン**: 条件に合致するファイルを検索
5. **ファイルを選択**: 収集したいファイルにチェックを入れる
6. **ファイルを収集**: 選択したファイルを収集先にコピー

## 除外設定について

除外フォルダは改行区切りで複数指定できます：

```
node_modules
.git
*.log
temp*
```

- 完全一致: `node_modules`
- ワイルドカード: `temp*` (tempで始まるフォルダ)

## 自動除外されるファイル

以下のファイルは自動的に除外されます：
- `.DS_Store` (macOS)
- `Thumbs.db`, `desktop.ini` (Windows)
- `.tmp`, `.log`, `.cache` 拡張子のファイル
- 隠しファイル（`.gitignore`、`.env`を除く）

## ビルド

アプリケーションをビルドする場合は、electron-builderなどのツールを使用してください。

```bash
npm install --save-dev electron-builder
```

package.jsonにビルドスクリプトを追加:
```json
{
  "scripts": {
    "build": "electron-builder"
  },
  "build": {
    "appId": "com.example.file-collector",
    "productName": "File Collector",
    "directories": {
      "output": "dist"
    }
  }
}
```
